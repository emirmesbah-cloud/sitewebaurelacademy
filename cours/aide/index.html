🔑 Accès & code

Mon code ne fonctionne pas, que faire ?

Vérifie que tu l'écris en majuscules et que tu inclus bien le tiret (ex: AU-A1B2C3). Si tu vois "Code invalide", recontacte-nous sur WhatsApp pour qu'on vérifie.

Le message "déjà utilisé sur un autre appareil" s'affiche

Ton code est lié à l'appareil où tu l'as activé la première fois. Si tu as changé de téléphone, contacte-nous sur WhatsApp avec ton code, on le débloque pour ton nouvel appareil.

J'ai perdu mon code, comment le récupérer ?

Envoie-nous ton nom complet et la date approximative d'achat par WhatsApp. On te renvoie ton code dans la journée.

Mon code peut-il expirer ?

Non. Ton code donne accès à vie aux cours. Il n'expire jamais.

📚 Cours & modules

La vidéo ne se lance pas

Essaye de : rafraîchir la page, changer de navigateur (Chrome recommandé), vérifier ta connexion internet. Si le problème persiste, envoie-nous un screenshot sur WhatsApp.

Comment marquer un module comme terminé ?

Clique sur le bouton "Marquer comme terminé ✓" en haut à droite de la page du module. Ta progression est sauvegardée automatiquement.

Quand mes bonus se débloquent-ils ?

Chaque bonus se débloque automatiquement après avoir terminé un module précis. Tu vois la condition de déblocage sur chaque bonus (ex: "Termine le module 1").

Je ne vois pas les Lives et le groupe WhatsApp

Ces fonctionnalités sont exclusives au pack Accompagné (code AC-). Si tu as acheté le pack Autonome (AU-), tu n'y as pas accès. Tu peux upgrader à tout moment — contacte-nous.

📱 Appareil

Je change de téléphone, que faire ?

Envoie-nous un message WhatsApp avec ton code actuel. On déverrouille le binding IP et tu peux réactiver ton code sur ton nouvel appareil.

Les vidéos fonctionnent-elles hors ligne ?

Les vidéos sont en streaming (VDOCipher) et nécessitent une connexion internet. Seuls les PDFs et audios bonus peuvent être téléchargés.

La plateforme marche sur mon téléphone ?

Oui. La plateforme est responsive et fonctionne sur tous les navigateurs mobiles récents (Chrome, Safari, Firefox). Il suffit d'aller sur cours.aurel-academy.com.

💳 Paiement & remboursement

Quelle est ta politique de remboursement ?

Satisfait ou remboursé sous 14 jours après activation. Envoie-nous un WhatsApp avec ton code et le motif, on fait le remboursement.

Puis-je avoir une facture ?

Oui. Demande-la par WhatsApp avec ton nom complet et ton adresse, on te l'envoie par email.

Je veux upgrader d'Autonome à Accompagné

C'est possible, tu ne paies que la différence. Contacte-nous sur WhatsApp pour activer l'upgrade.
