Questions fréquentes

On répond à tes questions

Comment j'obtiens un code d'accès ?

Tu achètes une formation sur aurel-academy.com/pflege/. Tu reçois ton code unique sur WhatsApp juste après le paiement.

Mon code ne marche pas, que faire ?

Vérifie qu'il commence bien par AU- ou AC-. Si le code est "déjà utilisé sur un autre appareil", contacte-nous sur WhatsApp — on peut le débloquer.

Je peux utiliser mon code sur plusieurs appareils ?

Non. Ton code est lié à ton appareil pour éviter le partage. Si tu changes de téléphone, contacte-nous pour transférer l'accès.

J'ai perdu mon code — je peux le récupérer ?

Oui. Envoie-nous un message WhatsApp avec ton nom et la date d'achat, on te renvoie ton code.

L'accès est vraiment à vie ?

Oui. Tu paies une seule fois. Tu gardes l'accès aux vidéos + bonus + mises à jour, à vie.
